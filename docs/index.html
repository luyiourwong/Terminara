下載 & 安裝

installation.sh
1

安裝 Python

確保你的系統已安裝 Python 3.8 或更新版本

python --version
2

下載遊戲

從 GitHub Release 下載最新版本

git clone https://github.com/your-repo/ai-terminal-game.git
3

安裝依賴

安裝必要的 Python 套件

pip install -r requirements.txt
4

開始遊戲

執行遊戲並開始你的冒險

python main.py

創建自訂世界！

想要在中世紀魔法世界中冒險？還是在賽博龐克未來都市中求生？


透過簡單的JSON配置檔案，你可以創造任何你想像得到的世界設定。定義角色、背景故事、可用物品、以及獨特的遊戲規則。


AI會根據你的世界設定，動態生成符合該世界觀的情境和對話，讓每個自訂世界都有其獨特的遊戲體驗。

world_config.json
{ "world_name": "賽博龐克 2087", "setting": "未來都市", "theme": "科幻反烏托邦", "player_role": "網路駭客", "starting_items": ["神經植入物", "解碼器"], "world_rules": [ "企業掌控一切", "AI監控無所不在", "反抗軍暗中行動" ] }
